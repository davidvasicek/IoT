## Regulace jasu 12V LED pásků

#### Komponenty

- Arduino deska
- Zdroj 12V
- MOSFET (např. P16NF06)
- LED pásek (např. SMD5050)

#### Postup

1. Zapojte obvod dle zapojení níže (~3 = PWM pin)
2. Spusťte software Arduino IDE.
3. Vložte kód do Arduino IDE a nahrejte jej do vaší desky Arduino.
